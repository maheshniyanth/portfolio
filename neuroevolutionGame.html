Neuroevolution Moble Game

Summary

The visual aspect is similar to most IO games like Deip.io. Mostly becuase my artistic ability is very limited. So, the character and enemies will be represented using basic geometric shapes. The background will consist of varying paper patterns (lined paper, graph paper).

The game consists of a 2D shooter survival in a procedurally generated maze. What makes this special is that the enemy entities spawn with varying ability levels and feautres (speed, damage, health). After every round a nueroevolution program (Combines feaures and adds mutations similar to actual evolution but a lot faster) is run with the remaining entities and a new, stronger set is spawned the next round. This repeats with every following round getting specificly harder for that player.
