Electromyography Based Instrument / Rythm Game

Introduction

I first saw the surface electromyography sensors used in a prostetic arm and I was amazed with the functionality that it could provide. Since then I have been interested in doing something with them. I also really love rythm games, so I came up with a concept for the first audio only rythm game using the EMG pads.

It works by using several EMG fitted into a wearable sleave that goes on the forearm. It detects the taps of each of the fingers. It then connects via Bluetooth to a device where the app would allow the user to select the song and calabrate the device (I dont have the time to learn the basics of app development for this project so I will just upload the programmed songs on to the sleave). The sound will then come from device.

The user will know what to do based on varying notes played in the audio indcating what finger to use as well as shorthand standing for a variety of things such as repeating a set note pattern or holding a note.

The main apeal of this would be its portability. The lack of screen allows the user to play it discreatly anywhere without looking away from what they are doing.

Research

I knew very little about these sensors other than the fact that they were impresise at tracking finger movement, so I had a lot of reasarch ahead of me.
