Multi-Directional Treadmill

A design concept for a treadmill that allows the user to move in all directions and inclines, simulating a virtual environment for Virtual reality or similar movement simulations. Using an internal roller design it allows powered movement in any direction. Screw drives use each of the support beams to create a stable 0% to 20% incline in any direction.
